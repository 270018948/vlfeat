Caltech-101 classification

This application uses VLFeat to train an image classifier on the Caltech-101 dataset. It uses:

  • PHOW features (dense multi-scale SIFT descriptors) as low level features;
  • the Elkan k-means algorotihm for fast visual word dictionary construction;
  • spatial histograms as image descriptors;
  • the homgeneous kernel map to train a Chi2 support vector machine (SVM) as a linear one;
  • PEGASOS for SVM training.

The program is fully containe into a single MATLAB M-file. The code is also bundled in VLFeat under the VLROOT/apps subdirectory.

The VLFeat open source library implements popular computer vision algorithms such as SIFT, MSER, k-means, hierarchical k-means, and agglomerative information bottleneck. It his written in C for efficiencly, but it is fully interfaced to MATLAB for ease of use, and it is thorughly documented. It supports Mac OS X, Linux, and Windows.

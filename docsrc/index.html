VLFeat is a cross-platform open source collection of vision algorithms with a special focus on visual features (SIFT, MSER) and clustering (k-means, hierarchical k-means, agglomerative information bottleneck). It bundles a MATLAB toolbox, a clean and portable C library and a number of command line utilities. Thus it is possible to use the same algorithm both from MATLAB, the command line, and your own programs.

VLFeat is an effort initiated by Andrea Vedaldi and Brian Fulkerson in 2007.

Getting started

VLFeat can be obtained here. To start using the MATLAB toolbox, you can read some of the tutorials and the list of MATLAB functions. To learn about the C library interface and details of the implemented algorithms, please refer to the C library documentation. Finally, to use the command line utilities, please refer to the relative manual pages.

VLFeat is an open source collection of vision algorithms with a special focus on visual features (SIFT, MSER, ...). It bundles a MATLAB toolbox, a clean and portable C library and a number of command line utilities. Thus it is possible to use the same algorithm both from MATLAB, the command line, and your own programs.

VLFeat is an effort initiated by Andrea Vedaldi and Brian Fulkerson in 2007.

Obtaining VLFeat

You can download the binary distribution (Mac, Linux, Windows) or the source code.

There are no special installation instructions: simply unpack the archive. To use the MATLAB Toolbox, open MATLAB, cd into the toolbox subdirectory and exectue the vlfeat_setup command. Use the overview_vlfeat command to print a summary of the toolbox contents.

Getting started

To start using the MATLAB toolbox, you can read the list of toolbox functions. To use the C library please refer to the C library documentation. Finally, to use the command line utilities, refer to the relative manual pages.

Notice that the authoritative description of the algorithms is contained in the C library documentation. Reading this material is generally not required to use the library from MATLAB or the command line, but it can be useful to understand the meaning of the algorithm parameters and have accurate informations on their implementations.
